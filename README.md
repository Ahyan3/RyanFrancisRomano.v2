## 🌟 Overview

This is my personal portfolio website showcasing my work as a **Full-Stack Developer, UI/UX Designer, and Computer Science Graduate**. Built with a cyberpunk/neon aesthetic featuring a 3D interactive background.

**Live Demo:** [ryanfrancisromanov2.netlify.app](https://ryanfrancisromanov2.netlify.app)

## 🎯 Features

### 🖥️ User Experience
- **3D Interactive Background** – Animated torus knot with floating particles
- **Full-Screen Carousel** – 1920x1080 optimized image gallery
- **Responsive Design** – Seamless experience on all devices
- **Smooth Scrolling** – Navigate between sections effortlessly
- **Glassmorphism Effects** – Cyber-themed cards with backdrop blur

### 📱 Sections
| Section | Description |
|---------|-------------|
| **Hero** | Terminal-style introduction with animated typing effect |
| **About** | Identity matrix + interactive profile photo carousel |
| **Projects** | Featured projects with gallery carousels |
| **Tech Stack** | Visual technology icons with hover descriptions |
| **Certifications** | Organized by year (2026, 2025, 2024) |
| **Leadership** | Student leadership roles and seminars |
| **Contact** | Formspree integration with cyber popup feedback |

### 🎨 Design
- **Cyberpunk/Neon Theme** – Cyan and purple gradients
- **Glitch Effects** – Border glitch animations
- **Tooltips** – Hover descriptions for technical skills
- **Animations** – Floating, pulsing, and gradient transitions

## 📦 Deployment

### Netlify
```bash
# Deploy via Netlify Drop
# Drag and drop the folder to: https://app.netlify.com/drop

# Or via Netlify CLI
npm install -g netlify-cli
netlify login
netlify deploy --prod
```

### GitHub Pages
```bash
git add .
git commit -m "feat: deploy portfolio"
git push origin main
```

---

## 🔧 Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ahyan3/RyanFrancisRomano.v2.git
   cd RyanFrancisRomano.v2
   ```

2. **Open in browser**
   ```bash
   # Open index.html directly
   start index.html
   # Or use Live Server in VS Code
   ```

3. **File Structure Requirements**
   - Ensure `images/` folder contains all assets
   - Check image paths match the HTML references
